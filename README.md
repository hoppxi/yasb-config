#  YASB Theme Config

```css
:root {
	--blush-petal: rgb(248, 222, 218);   /* rosewater → blush-petal */
	--sunset-peach: rgb(243, 208, 195);  /* flamingo → sunset-peach */
	--orchid-bloom: rgb(240, 198, 226);  /* pink → orchid-bloom */
	--violet-mist: rgb(190, 160, 230);     /* mauve → violet-mist */
	--coral-glow: rgb(236, 120, 146);    /* red → coral-glow */
	--rosewood: rgb(221, 145, 157);      /* maroon → rosewood */
	--apricot-haze: rgb(247, 176, 128);  /* peach → apricot-haze */
	--amberlight: rgb(245, 215, 160);    /* yellow → amberlight */
	--moss-green: rgb(150, 210, 150);    /* green → moss-green */
	--seafoam: rgb(134, 211, 198);       /* teal → seafoam */
	--clear-sky: rgb(125, 205, 225);     /* sky → clear-sky */
	--coastal-blue: rgb(105, 185, 220);  /* sapphire → coastal-blue */
	--morning-blue: rgb(125, 170, 240);  /* blue → morning-blue */
	--lavender-field: rgb(165, 180, 250);/* lavender → lavender-field */

	--cloud-text: rgb(215, 222, 245);    /* text → cloud-text */
	--mist-text: rgb(195, 202, 230);       /* subtext1 → mist-text */
	--fog-text: rgb(175, 182, 210);        /* subtext0 → fog-text */

	--soft-overlay: rgb(155, 160, 190);    /* overlay2 → soft-overlay */
	--mid-overlay: rgb(135, 140, 170);     /* overlay1 → mid-overlay */
	--deep-overlay: rgb(115, 120, 145);    /* overlay0 → deep-overlay */

	--twilight-surface: rgba(95, 98, 120, .5);  /* surface2 → twilight-surface */
	--evening-surface: rgba(75, 78, 100, .5);   /* surface1 → evening-surface */
	--night-surface: rgba(50, 52, 70, .5);      /* surface0 → night-surface */

	--dusk-base: rgba(38, 38, 54, .5);          /* base → dusk-base */
	--midnight-mantle: rgba(28, 28, 42, .5);    /* mantle → midnight-mantle */
	--noir-crust: rgba(20, 20, 30, .5);         /* crust → noir-crust */
}
```
